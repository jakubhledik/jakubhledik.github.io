# Semafor

PWA aplikace, která zobrazuje semafor se dvěma světly. Funguje offline, lze nainstalovat na domovskou obrazovku.

**[→ Otevřít](https://jakubhledik.github.io/semafor)**

## Režimy

| Režim | Chování |
|-------|---------|
| **Odpočet** | Klepnutí spustí 3sekundový odpočet se zvukem, pak přepne na zelenou |
| **Ručně** | Okamžité přepínání červená ↔ zelená |

Druhý přepínač mění barvu pozadí: **Tmavý** (výchozí) / **Světlý**.

## Technologie

Vanilla JS, CSS, Web Audio API, Service Worker — žádné závislosti.
